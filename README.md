# Neon Void

A dark VS Code theme with neon yellow-green, cyan, and pink accents, based on Xcode 11 Default Dark.

## Install

Search for **Neon Void** in the Extensions view (`Cmd+Shift+X`), or run:

```
ext install CanedCat.neon-void-theme
```

Then select it via **Preferences: Color Theme** (`Cmd+K Cmd+T`).

## Features

- Neon accent palette across editor UI, status bar, and activity bar
- Bracket pair colorization tuned to match the neon accents
- Semantic highlighting support
